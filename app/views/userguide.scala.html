Resource Locator User Guide

Resource Locator is an application designed to help contact center counselors retrieve information and referrals so they can provide better service to call-in customers.

Through a user-friendly interface, the counselor first enters the client's location and AIRS needs classification.

Location can be provided either by entering the complete address, the zip code only, or as the client’s GPS coordinates (either manually or by right clicking over the map at the desired location). To edit: click on the location field (doing so will highlight the previous entry in blue); then write over with a new location. Once this field is completed a red marker will appear on the map at the correct location.

Clicking on the Needs field displays a list of needs. A) Just click to select one need; B) click while pressing down Ctrl to select more than one need. If more than one need, click outside the area to complete selection and perform Search. (You can always delete a need by clicking on the little x on the left of each need, select a new need, or add others. In each case, however, click again on Search.)

With just a click on the Search button, Resource Locator generates a blue marker on the map with the locations of service providers for the particular need(s) identified, sorted by distance from the customer's stated location. Inside each marker, the number represents the order, by distance, to the user’s location (represented by the red marker).

In order for counselors to more quickly, easily, and accurately retrieve all the information requested by the client, the search can be further narrowed through four basic filters:

Filters:

  1. Distance: The system searches within a circle centered at the GPS position reported by the client, and displays the results within the selected radius in miles. The default is five miles.
  2. Needs: The system searches only for those service providers that satisfy the number of needs selected on this filter.
  3. Providers: The map generated shows only the requested number of service providers closest to the reference provided by the client.
  4. Target Population: Particular populations have assigned service providers. When one of these populations is identified, the system displays only those providers assigned to that population.

Search result:

Clicking on the Search button will display results as a map on the right of the screen and as a list on the left (below the Search button). All results are numbered by distance from the client’s stated location. Scrolling down the list and clicking on the Full Info button for each provider reveals all available information for this provider. Left clicking directly on the blue map markers displays an infowindow with basic provider information.

Specifications:

  • If the Distance filter is deactivated, no search circle is displayed and the system shows all providers for all selected needs, regardless of distance.
  • When no provider meets the search criteria, an empty result is generated.
  • When any search criteria or filter is changed, you must update the search.
  • The red marker on the map indicates the client's stated location, from which distances are measured linearly.
  • Blue markers for your search are numbered and sorted by distance, measured from the client's stated location to the provider.
  • A jump in numbering within markers indicates that two or more providers have the same location. These can be identified in the search result list displayed below the search button.
  • The Reset button clears all fields to start a new search.
  • The Logout button on the top right closes the application and returns to the logged page.

Technical Specifications:

  • Google Chrome is recommended. (Due to Google's technology development, some items are not compatible with Internet Explorer.) Resource Locator has also been tested and works with Mozilla Firefox.
